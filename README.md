## 🎯 怎么读这个教程

**完全零基础** → 按顺序从 [第一章](./tutorial/01-认识终端和命令行.md) 读到第八章，每章都动手操作

**会用终端，但没用过 Claude Code** → 从 [第二章](./tutorial/02-Claude-Code是什么.md) 开始读，[第三章](./tutorial/03-安装Claude-Code.md) 安装，[第四章](./tutorial/04-第一次运行.md) 运行

**已经在用，想深入掌握** → 直接跳到 [第五章](./tutorial/05-核心概念.md)、[第六章](./tutorial/06-Skills完全指南.md)、[第七章](./tutorial/07-实用能力教程.md)

**只想搞清楚怎么配置 API 和付钱** → 直奔 [第八章](./tutorial/08-第三方模型和计费.md)

**一知半解，想了解 AI Agent** → 跳到 [第二章：Claude Code 是什么 — AI Agent 概念对比](./tutorial/02-Claude-Code是什么.md)

- **风格**：每个概念先用生活类比解释，再讲技术细节

- **术语**：所有专业名词用 **粗体** 标出，括号里是白话解释
- **参考**：基于 [Claude Code 官方文档](https://code.claude.com/docs/en/overview) 编写
