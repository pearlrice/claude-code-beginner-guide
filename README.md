# Claude Code 新手全方位教程

> 从零开始，学会使用 Claude Code 命令行版。面向纯新手，每步配有图示和说明。

## 这是什么？

一份面向**完全零基础**用户的中文教程，教你如何使用 Claude Code —— 一个住在终端里的 AI 编程助手。

不需要编程经验。你只需要会用电脑、会打字。

## 怎么读

- **完全零基础** → 从第一章按顺序读，每章动手操作
- **会用终端，没用过 Claude Code** → 从第二章开始
- **已经在用，想深入** → 跳第五、六、七章
- **只想搞清楚怎么付钱** → 直奔第八章
